## 什么是「逗号」？

**逗号** 是一个轻量级的 macOS 菜单栏休息提醒工具。它的核心理念是：**在你持续工作的过程中，适时插入一个「逗号」——停顿片刻，起身活动、喝水、远眺，再继续前行。**

无论你是程序员、设计师、作家还是任何需要长时间面对屏幕的人，逗号都能帮助你养成定时休息的习惯，减少眼疲劳和久坐伤害。

- 🪶 **极致轻量**：纯 SwiftUI & AppKit 原生开发，无任何第三方依赖，内存占用极小
- 🧭 **常驻菜单栏**：不占 Dock 栏空间，点击菜单栏的「，」图标即可操作
- 🧠 **智能提醒**：支持心流模式、渐进式提醒、自动恢复等高级功能

## 详细功能介绍

### ⏱ 自定义倒计时间隔

你可以自由设定计时器的分钟和秒数，精确到秒。默认间隔为 60 分钟，可调节范围 0~999 分 + 0~59 秒。

> 适合你的节奏：番茄钟（25分钟）、每小时休息（60分钟）、两小时长间隔（120分钟）……完全自定义。

---

### 🔔 自定义提醒文字

弹出窗口显示的是**你写的文字**，而不是固定的提示语。你可以设为：

- 「起来走走，看看窗外」
- 「喝水时间！」
- 「闭眼休息 30 秒」

提醒语会持久保存，下次启动应用时自动恢复。

---

### 🌊 心流模式（Flow Mode）

> **灵感来源**：当你进入心流状态时，一个突然弹出的窗口会粗暴地打断你的思路。

心流模式解决了这个问题：

1. 倒计时到达后，**屏幕四边会出现蓝色渐变光晕**（正弦波脉冲动画，持续 2.4 秒），作为温和的视觉提醒
2. 你**无需做任何事情**——计时器会自动重置，继续下一轮倒计时
3. 连续闪烁达到你设定的次数后（默认 3 次），**才会弹出提醒窗口**

这样，当你正处于心流中，蓝色光晕只是轻触你的边缘视觉，提醒你「时间差不多了」；而当你确实需要强制休息时，弹窗才会出现。

- **可配置闪烁次数**：1 ~ 99 次，按你的专注节奏调整

---

### 📐 渐进式提醒（Progressive Reminder）

渐进式提醒让你的休息提醒**越来越频繁**，模拟身体逐渐累积的疲劳感。

开启后，每次提醒结束后，下一次倒计时的间隔会自动缩短。提供两种策略：

#### 每次减半（Halve）

每次提醒后，剩余间隔减半。例如初始间隔 60 分钟：

```
第 1 次提醒 → 60 分钟后
第 2 次提醒 → 30 分钟后
第 3 次提醒 → 15 分钟后
第 4 次提醒 → 10 分钟后（达到最小间隔限制）
```

#### 固定减少（Fixed）

每次提醒后，间隔减少固定时长（由你设定）。例如初始 60 分钟，每次减少 10 分钟：

```
第 1 次提醒 → 60 分钟后
第 2 次提醒 → 50 分钟后
第 3 次提醒 → 40 分钟后
...直到达到最小间隔
```

- **最少间隔保护**：可设定一个最小间隔（分钟+秒），提醒不会比这个更频繁
- **手动停止后重置**：如果你主动停止了计时，下次启动会恢复到完整的初始间隔
- **与心流模式兼容**：两种模式可以同时开启

---

### 🔗 绑定 App 自动开始计时

你可以绑定任意 macOS 应用程序（.app），当你打开该 App 时，逗号会**自动开始计时**。

- 点击「绑定App (自动开始)」，选择 `/Applications` 下的任意应用
- 例如：绑定 VS Code → 每次打开 VS Code 自动开始计时
- 可随时更改绑定或取消绑定

> 适合固定工作流的用户：打开工作软件 = 开始专注计时，无需手动操作。

---

### 🔄 继续时自动重启计时

在提醒弹窗中点击「继续」后，自动开始新一轮的倒计时。省去再次手动点击「开始计时」的步骤。

> 这个功能与「休息时检测活动」互斥，只能同时启用一个。

---

### 🕵 休息时检测活动（Auto Resume）

这是一个**智能自动恢复**功能：当你结束休息、返回电脑开始操作时，逗号会自动检测到并重新开始计时。

工作流程：

1. 你在提醒弹窗中点击「我知道了」→ 进入休息状态
2. **等待 10 秒**后，逗号开始监听鼠标和键盘输入
3. **第一次检测**到活动 → 屏幕边缘闪烁蓝色光晕（提示"你回来了！"）
4. 再等待 **10 秒**，进行**第二次检测**
5. 如果再次检测到活动 → **自动开始计时**，菜单栏短暂显示「▶ 已开始计时」
6. 如果期间一直没有活动，则保持休息状态

> 两次检测 + 光晕提示的设计，确保你是真的回到电脑前才开始计时，而非误触鼠标。

---

### 📊 今日统计

在主界面底部显示当日累计数据：

- **今日专注时长**：计时器运行期间的总时长，精确到分钟
- **今日休息次数**：今天触发了多少次提醒

统计会在每天午夜自动归零重置，无需手动操作。

---

### 💡 菜单栏状态显示

逗号在 macOS 菜单栏的图标会随状态变化，让你一眼就知道当前状态：

| 状态 | 显示 | 含义 |
|------|------|------|
| 空闲 | `，` | 未开始计时，等待你的指令 |
| 运行中 | `⏳ 01:23:45` | 正在倒计时，实时显示剩余时间 |
| 提醒中 | 🔔 | 时间已到，弹出提醒窗口 |


## 使用指南

1. 点击菜单栏的 **「，」** 图标，展开设置面板
2. 设置倒计时间隔（分 + 秒）
3. （可选）输入自定义提醒文字
4. （可选）开启心流模式、渐进式提醒、绑定 App 等高级功能
5. 点击 **▶ 开始计时**
6. 时间到后弹出提醒窗口：
   - 点击 **我知道了** → 确认并消除提示音，然后可选择「继续」或「结束」
   - 点击 **继续** → 关闭窗口，根据设置决定是否自动重启计时
   - 点击 **结束** → 关闭窗口，回到空闲状态
7. 随时可以点击 **取消** 中止当前计时

> 💡 提示：点击面板中的 **❓** 图标可以查看内联的功能说明。

## 技术架构

| 层面 | 技术 |
|------|------|
| UI 框架 | SwiftUI |
| 系统集成 | AppKit（NSStatusBar、NSWindow、NSPopover） |
| 事件监听 | CGEvent（鼠标/键盘活动检测） |
| 动画 | Core Animation（心流模式屏幕边缘光晕） |
| 数据持久化 | UserDefaults（设置与统计） |
| 外部依赖 | **无** |

### 代码结构

```
douhao/
├── Info.plist          # 应用配置（Bundle ID、版本、系统要求等）
└── Sources/
    └── main.swift      # 全部源码（单文件）
```

**主要模块：**

- `ProgressiveMode` — 渐进式提醒的两种策略枚举（减半 / 固定减少）
- `TimerViewModel` — 核心业务逻辑：计时器管理、状态机、设置持久化、统计数据
- `PopoverContent` — 菜单栏弹出面板的 UI 层（空闲/运行/提醒三种状态视图）
- `ProgressiveSettingsView` — 渐进式提醒的详细设置子面板
- `AlertWindowController` / `AlertContentView` — 提醒弹窗（确认 → 显示休息时长 → 继续/结束）
- `StatusBarController` — 菜单栏管理：状态显示、弹窗协调、App 绑定监听、心流光晕动画、休息后活动检测
- `AppDelegate` — 应用入口，设置菜单栏模式（隐藏 Dock 图标）
