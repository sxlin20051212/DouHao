# 逗号

macOS 菜单栏定时提醒工具，帮你养成规律休息的习惯。

## 功能

- **定时提醒**：自定义间隔（分/秒），到时间后弹窗提醒休息
- **心流模式**：倒计时到后先闪烁屏幕边缘 N 次，不打断心流，达到次数后才弹窗
- **渐进式提醒**：每次提醒后自动缩短间隔（减半或固定减少），让提醒越来越频繁
- **句号模式**：强硬打断机制。开启后需先绑定一个 App，倒计时到后弹出 5 秒倒计时，到时间自动强制关闭绑定的 App
- **绑定 App**：绑定任意 macOS 应用，该应用启动时自动开始计时
- **休息时检测活动**：结束休息后检测用户活动，自动恢复计时
- **继续时自动重启计时**：点击继续后自动开始新一轮倒计时
- **静音提醒**：静音模式下提醒不会播放声音，仅弹窗和闪烁

## 安装

1. 下载 `，.dmg` 并打开
2. 将 `，.app` 拖入 Applications 文件夹
3. **首次打开**：右键点击 `，.app` → 选择「打开」→ 在弹出的对话框中点击「打开」
   - 如果仍然提示"文件已损坏"，在终端执行：`xattr -cr /Applications/，.app`
